# Full Stack Open - University of Helsinki

This repository contains my exercises, notes, and project work completed while following [Full Stack Open](https://fullstackopen.com/en), an open online course by the University of Helsinki.

Full Stack Open focuses on modern JavaScript-based web development. The course covers building single page applications with React, using REST APIs with Node.js, and introduces topics such as Redux, MongoDB, GraphQL, TypeScript, testing, CI/CD, containers, and relational databases.
