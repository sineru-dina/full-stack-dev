# Part_0 - Fundamentals of Web Apps

This folder contains my work for Part 0 of the University of Helsinki's Full Stack Open course.

Part_0 introduces the fundamentals of web applications, including how the browser and server communicate, how HTTP requests work, and how traditional web applications differ from single page applications.

## Exercises Completed

The work in this folder is focused on sequence diagrams that describe browser-server communication.

| Exercise | Description |
|---|---|
| 0.4 | New note diagram for the traditional notes application |
| 0.5 | Single page application diagram |
| 0.6 | New note diagram for the single page application |

## Topics Practiced

- Browser-server communication
- HTTP GET and POST requests
- HTML document loading
- CSS and JavaScript loading
- Form submission behavior
- Single page application behavior
- Sequence diagrams

## Notes

These exercises helped me understand how web applications communicate behind the scenes, especially the difference between traditional page reload-based applications and modern single page applications.
